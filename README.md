# Calendar Booking API

##  Live URL
https://calender-booking-api-1.onrender.com

##  Features
- Create booking
- Prevent double booking (conflict rejection)
- Get all bookings
- Date & time validation

##  Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

##  API Endpoints

### Create Booking
POST /bookings

### Get All Bookings
GET /bookings

##  Conflict Logic
The system checks:
- Same date
- Overlapping time slots
